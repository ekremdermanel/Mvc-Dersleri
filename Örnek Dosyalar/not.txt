Kullanılan layout ta doğrulama kontrolleri javascript leri varsayılan olarak eklenmemiş.
js dosyaları Views - Shared - _ValidationScriptsPartial.cshtml dosyasında çağırılmış.
Doğrulama yapılacak formlarda bu partial ın Layout'daki Scripts section ında çağrılması gerekiyor.



Soru2.cshtml view dosyasına ekle

@section Scripts {
    <partial name="_ValidationScriptsPartial"></partial>
}
